### How to use it

After cloning the repository, users can install the dependencies with:

pip install -r requirements.txt

These libraries are used for:

scapy → packet sniffing and network analysis

numpy → numerical processing

matplotlib → graph/traffic visualization

pandas → traffic data handling and logs
