# AetherEye – Network Monitoring Application

AetherEye is a lightweight **Python-based network monitoring application** that captures and analyzes real-time network traffic.
The project demonstrates basic **packet sniffing, protocol analysis, and traffic visualization** using Python libraries such as **Scapy** and **Matplotlib**.

---

## 🚀 Features

* Capture real-time network packets
* Analyze packet protocols (TCP, UDP, ICMP, etc.)
* Monitor network traffic activity
* Visualize network data using graphs
* Lightweight and easy to run

---

## 🛠️ Technologies Used

* Python
* Scapy
* NumPy
* Matplotlib
* Pandas

---


## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/AetherEye.git
```

2. Navigate to the project folder

```bash
cd AetherEye
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

Run the main script to start monitoring network traffic:

```bash
python main.py
```

The program will start capturing packets and displaying traffic statistics.
