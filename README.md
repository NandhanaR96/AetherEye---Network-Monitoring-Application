# AetherEye – Network Monitoring Application

AetherEye is a lightweight real-time network monitoring tool developed using Python.  
It captures live network packets, analyzes traffic patterns, and visualizes network statistics to help administrators understand network behavior.

## Features

- Live packet capture using Scapy
- Protocol identification (TCP, UDP, ICMP)
- Real-time traffic analysis
- Bandwidth usage monitoring
- Graphical visualization using Matplotlib
- Alert generation for unusual traffic patterns

## Technologies Used

- Python
- Scapy
- NumPy
- Matplotlib

## System Requirements

- Python 3.8+
- 4GB RAM minimum
- Windows or Linux
